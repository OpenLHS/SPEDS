5.1. [CT_001] Cas de Test 1 - Échec de l’envoi de données avec un IRI destinataire non rejoignable

Description

Échec de l’envoi de données avec IRI destinataire non rejoignable sur le protocole de communication.

Hypothèse

S.O.

Données d’entrée
Identifiant Valeur Description

E_01

InterfaceDataUnit-5_6 ` où le service est `transfer et la primitive de service est request.

Un IDU avec un IRI destinataire qui n’existe pas dans la configuration donnée.
Données de sortie
Identifiant Valeur Description

S_01

InterfaceDataUnit-5_6 où le service est transfer, la primitive de service est confirm et ConfirmMessageContent contient la valeur FAILED.

  1. L’IDU sortant du processus correspond aux données de sorties.

Données de sortie
Identifiant Valeur Description

S_02
